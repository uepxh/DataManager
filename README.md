# DataManager

本项目是一个汇编语言课程设计题目的源代码，采用Win32汇编实现了一个简单的学生信息管理系统

具有最基本的增删改查功能

支持将数据保存到本地

也能载入本地保存的数据

| 文件名          | 意义                           |
| --------------- | ------------------------------ |
| DataManager.asm | 源代码                         |
| DataManager.inc | 头文件，包含宏和一些常量的定义 |
| DataManager.rc  | 资源文件脚本                   |

DataManager这个文件夹是一个RadAsm工程文件夹

使用的RadAsm版本为2.2.1.9
